# Jellyfin Launch Potplayer

Tampermonkey userscript that adds external player buttons to Jellyfin Web.

## Features
- Adds Potplayer and Infuse launch buttons on item details pages.
- Supports Jellyfin Web routing changes for item id parsing.
- Uses external subtitle selection when available.

## Install
1. Install Tampermonkey in your browser.
2. Open the userscript URL:
   https://raw.githubusercontent.com/Unarmored7/jellyfinLaunchPotplayer/main/jellyfinLaunchPotplayer.user.js
3. Confirm installation in Tampermonkey.

## Usage
- Open a media item detail page in Jellyfin Web.
- Click the external player button.
